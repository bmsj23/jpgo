## 1. Design System Overview

### 1.1 Color Palette

| Color | Hex | Usage |
|-------|-----|-------|
| **Jeepney Yellow** | `#FFB800` | Primary brand color, CTAs, active states |
| **Deep Blue** | `#1A237E` | Secondary color, headers, text emphasis |
| **Success Green** | `#4CAF50` | Available seats, online status, success |
| **Warning Orange** | `#FF9800` | Limited seats, syncing status |
| **Error Red** | `#F44336` | Full seats, offline, errors |
| **Background Light** | `#F5F5F5` | App background (light mode) |
| **Surface White** | `#FFFFFF` | Cards, bottom sheets, inputs |
| **Text Primary** | `#212121` | Headings, primary text |
| **Text Secondary** | `#757575` | Captions, secondary info |

| **Small** | Inter | 12sp | Regular | Timestamps, hints |
