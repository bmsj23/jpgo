                           ARCHITECTURE: WEB PWA
================================================================================

Platform Strategy: 100% Web Progressive Web Apps (PWAs)
- Driver App:    PWA (Safari/Chrome on iPhone/Android)
- Commuter App:  PWA (Any browser on any device)
- Admin Dashboard: Web App (Desktop browsers)

Why PWA?
- Zero cost: No Apple Developer ($99/yr) or Google Play ($25) fees
- No Mac required: Build and test on Windows
- Cross-platform: Works on iOS Safari, Android Chrome, Desktop
- Installable: Add to Home Screen for native-like experience
- Offline capable: Service workers for caching

Hosting: Firebase Hosting (Free Tier - 10GB/month)

                         QUICK START COMMANDS
================================================================================

# run driver app in chrome:
cd apps/driver_app && flutter run -d chrome

# run commuter app in chrome:
cd apps/commuter_app && flutter run -d chrome

# run admin dashboard in chrome:
cd apps/admin_dashboard && flutter run -d chrome

# test on iphone (same wifi network):
cd apps/driver_app
flutter run -d web-server --web-hostname 0.0.0.0 --web-port 8888
# then open http://10.21.106.82:8888 in safari on iphone

# build for production:
flutter build web --release

# deploy to firebase hosting:
firebase deploy --only hosting
